# 射魂长弓技能系统

## 功能说明

这是一个完整的按键绑定技能系统，实现了"射魂长弓"技能。

### 技能特性

- **按键绑定**: 默认使用 `X` 键触发（可在游戏设置中修改）
- **技能名称**: 射魂长弓
- **蓄力时间**: 2秒
- **蓄力效果**: 玩家周围会显示螺旋上升的粒子效果
- **发射效果**: 发射一支带有粒子轨迹的强力箭矢
- **箭矢属性**: 
  - 伤害: 10.0
  - 暴击箭
  - 穿透等级 3

## 文件结构

### 客户端文件
- `KeyBindings.java` - 定义按键绑定
- `ClientEvents.java` - 注册按键映射
- `KeyInputHandler.java` - 处理按键输入事件

### 网络文件
- `ModNetworking.java` - 网络通道管理
- `SoulBowPacket.java` - 客户端到服务端的数据包

### 技能逻辑
- `SoulBowSkill.java` - 技能实现（蓄力、粒子效果、箭矢发射）

### 语言文件
- `zh_cn.json` - 中文翻译
- `en_us.json` - 英文翻译

## 使用方法

1. 进入游戏
2. 按下 `X` 键（或在设置中修改的按键）
3. 等待2秒蓄力（会显示粒子效果和聊天消息）
4. 自动发射带有粒子轨迹的强力箭矢

## 修改按键绑定

在游戏中：
1. 打开设置 (ESC)
2. 选择 控制 (Controls)
3. 找到 "MCNB 技能" 分类
4. 点击 "射魂长弓" 并按下你想要的按键

## 自定义粒子效果

在技能代码中使用了两种粒子效果：
- `mcnb:bow_charge` - 蓄力粒子效果
- `mcnb:soul_arrow` - 箭矢轨迹粒子效果

你需要在 `assets/mcnb/effeks/` 目录下创建对应的粒子效果文件。

## 技术说明

- 使用 Forge 网络系统实现客户端到服务端通信
- 使用线程实现延迟执行和持续粒子效果
- 防止重复触发（蓄力期间无法再次触发）
- 所有逻辑在服务端执行，确保多人游戏同步
