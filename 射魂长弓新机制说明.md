# 射魂长弓技能 - 新机制说明

## 🎯 新的使用方式

之前的机制是：按下 X 键 → 自动蓄力2秒 → 自动发射箭矢

**现在改为更灵活的机制：**
1. 按下 **X 键** 激活技能（显示"§e射魂长弓已就绪！下一次拉弓将触发技能"）
2. 手持弓箭，正常拉弓射击
3. 这次射出的箭矢会自动应用技能效果（高速、高伤害、粒子轨迹）
4. 技能效果只对下一次射箭有效，用完后需要重新按 X 键激活

## ✨ 新机制的优点

- **更灵活**：你可以控制何时使用技能
- **可取消**：激活后再次按 X 键可以取消
- **省资源**：只有在你真正需要时才使用技能
- **兼容性好**：可以配合任何弓使用（普通弓、附魔弓、其他MOD的弓）

## 🎮 完整使用流程

1. **激活技能**
   - 按下 X 键
   - 听到经验球拾取音效
   - 看到提示："§e射魂长弓已就绪！下一次拉弓将触发技能"

2. **射箭**
   - 拿出弓箭
   - 正常拉弓瞄准
   - 松手射击

3. **技能触发**
   - 箭矢速度变为设定值（默认3.0）
   - 箭矢伤害变为设定值（默认10.0）
   - 箭矢获得暴击和穿透效果
   - 箭矢带有炫酷的粒子轨迹
   - 播放末影龙射击音效
   - 玩家周围出现环形粒子效果

4. **再次使用**
   - 技能效果消耗完毕
   - 需要重新按 X 键激活

## 🔧 如何自定义箭矢属性

在 `Mcnb.java` 的 `commonSetup` 方法中添加：

```java
private void commonSetup(final FMLCommonSetupEvent event) {
    // ...existing code...
    
    // 自定义射魂长弓属性
    event.enqueueWork(() -> {
        SoulBowSkill.setArrowSpeed(5.0);    // 箭矢速度
        SoulBowSkill.setArrowDamage(25.0);  // 箭矢伤害
        SoulBowSkill.setArrowPierceLevel(5); // 穿透等级
        SoulBowSkill.setArrowCrit(true);    // 是否暴击
    });
}
```

## 📊 推荐配置方案

```java
SoulBowSkill.setArrowSpeed(8.0);    // 超快速度
```

```java
SoulBowSkill.setArrowPierceLevel(0); // 不穿透
```

```java
SoulBowSkill.setArrowSpeed(4.0);     // 正常速度

SoulBowSkill.setArrowPierceLevel(10); // 超高穿透
```

## ❓ 常见问题

**Q: 激活技能后不小心按了两次怎么办？**
A: 再次按 X 键会取消激活，显示"§c射魂长弓已取消"

**Q: 激活后忘记用了怎么办？**
A: 激活状态会一直保持，直到你射箭或手动取消

**Q: 可以用在弩上吗？**
A: 目前只支持弓，因为监听的是箭矢生成事件

**Q: 技能有冷却时间吗？**
A: 没有冷却时间，用完立即可以再次激活

## 🎨 视觉效果

- **激活时**：播放经验球音效
- **射箭时**：
  - 玩家周围环形粒子爆发
  - 播放末影龙射击音效
  - 箭矢带有持续5秒的粒子轨迹
